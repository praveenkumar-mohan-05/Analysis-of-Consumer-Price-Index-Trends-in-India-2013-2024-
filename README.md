# Analysis-of-Consumer-Price-Index-Trends-in-India-2013-2024-
A Python-based data analysis project using India’s CPI dataset from 2013–2024. The project explores inflation trends across Rural, Urban, and Combined sectors through data cleaning, EDA, and visualizations using Pandas, NumPy, Matplotlib, and Seaborn to generate meaningful economic insights.


Data Source:
1. Source : Dataset collected from Indian Government Portal

2. Link : https://www.data.gov.in/resource/all-india-consumer-price-index-ruralurban-upto-january-2024

3. Location : Rural/Urban ( Pan India )

4. Year / Timeline : January 2013 to January 2024 ; Published On: 05/06/2024

5. Domain : Statistical [Ministry of Statistics and Programme Implementation]
